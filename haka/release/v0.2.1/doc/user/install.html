2. Installation¶

2.1. Download the package¶

First, download the Debian packages from the Haka website in the Download section.

See also

If you need a package for another distribution, you will need to build it from the sources. See the section Build for more details about this.

2.2. Install¶

For each package that you want to install, type the following command:

$ sudo dpkg -i <package name>.deb
