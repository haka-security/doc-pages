Packet manipulation utilities.

Enums

packet_status enum

Packet status.

Values:

  • STATUS_NORMAL -

    Packet captured by Haka.

  • STATUS_FORGED -

    Packet forged.

  • STATUS_SENT -

    Packet already sent on the network.

packet_mode enum

Packet capture mode.

Values:

  • MODE_NORMAL -

    Normal mode (read/write).

  • MODE_PASSTHROUGH -

    Listen mode (read-only).
